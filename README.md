# 停车场管理系统 java353

## 项目概述

停车场管理系统是一款基于B/S架构和MVC设计模式的Web应用。该系统实现前后端分离，前端采用Vue框架，后端使用Java和SpringBoot技术栈，致力于提供高效便捷的停车场管理解决方案。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**：IDEA、JDK1.8、Maven
- **数据库**: Mysql
- **技术语言**: Java
- **后端框架**: SpringBoot
- **持久层框架**: Mybatis-Plus
- **前端框架**: Vue

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 系统角色

- 普通用户：具有基础操作权限，可管理个人信息及停车位预定缴费。
- 管理员用户：拥有系统管理权限，能够进行用户管理、车辆管理及停车管理。

## 运行环境

- **操作系统**: Windows

## 部署步骤

1. 配置开发环境，安装IDEA、JDK1.8、Maven。
2. 创建Mysql数据库，导入初始数据。
3. 使用IDEA打开项目，并配置相应的数据库连接信息。
4. 编译项目，通过Maven命令行或IDEA工具。
5. 部署到Web服务器。

## 核心亮点

- 前后端分离，易于维护和扩展。
- 使用SpringBoot快速构建后端服务，提高开发效率。
- 采用成熟的技术栈，保障系统的稳定性和安全性。
